How to use UI:

1. Select Farm in TreeView Window
2. Enter name of Item/Container in specified window.
3. Select either add item or add container depending on user preference
4. To change a component's name, loaction, dimensions or price, the desired item/container must first be selected on the TreeView.
   Then you can enter specified changes in given textbox, after that, you may hit the button of the desired process. 
5. Status messages within the UI (and in eclipse console) will show error or successful messages for processes.
